# To do list

A full stack web app, to create lists, tasks and tags.

- [Frontend roadmap](/frontend/todo.txt)
- [Backend roadmap](/backend/todo.txt)

### Stack

    Backend: Nest, postgres, prisma, graphql, jwt, bcrypt, jest, supertest-graphql, docker
